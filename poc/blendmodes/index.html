Proof-of-Concept - Projekt I - Blendmodes

Untersuchung verschiedener Blendmodes für etwaigen Erkenntnisgewinn für den Einsatz in Projekt I. Die Blendmodes werden per CSS durchgeführt. Mit der CSS Eigenschaft mix-blend-mode lassen sich Bilder mit dem Hintergrund überblenden. Der folgende Aufbau demonstriert die verschiedenen Blendmodis. Jeder Bereich weißt das Originalbild als CSS Eigenschaft background-image auf. Daneben werden die vier Aufnahmetypen als Bild (per img Tag) eingebunden. Diese erhalten den jeweiligen Blendmod per CSS-Klasse.
